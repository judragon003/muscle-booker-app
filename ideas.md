# 🏋️ 肌肉書僮風控儀表板 - 設計哲學與品牌定位

## 📐 設計方向選擇

### 三大風格方案

#### 1️⃣ 「鋼鐵戰士」- 深色科技風 (Probability: 0.08)
**簡介**：黑色背景 + 霓虹綠/紅燈號，模擬交易終端機與駭客風格，強調數據密度與即時性。

#### 2️⃣ 「禪意交易」- 極簡白色風 (Probability: 0.05)
**簡介**：純白背景 + 灰色漸層，極簡主義，強調清晰度與專注力。

#### 3️⃣ **「肌肉鐵血」- 深灰 + 金色銳角風** ✅ **選定方案** (Probability: 0.07)
**簡介**：深灰色背景 + 金色/橙色強調色，融合現代商務與運動品牌美學，傳達「力量、決斷、專業」的氣質。

## 🎨 選定方案：「肌肉鐵血」深度定義

### 🏛️ 設計運動
**Modern Brutalism meets Sports Brand Aesthetics**  
融合粗獷的幾何線條、沉穩的色彩、銳利的排版，呈現「交易不是賭博，是科學與紀律」的品牌核心。

### 💪 核心原則 (Core Principles)
1. **數據優先** - 所有視覺元素服從資訊層級，絕不裝飾優先
2. **銳利決斷** - 斜線、銳角、粗體字型傳達果決的交易風格
3. **深度對比** - 深灰背景 + 金色強調 + 白色文字，確保高可讀性
4. **動態即時感** - 微動畫與漸變傳達市場的實時脈動

### 🎨 色彩哲學
- **主背景**：`oklch(0.15 0.01 280)` - 深灰藍，沉穩專業
- **強調色（主力買超）**：`oklch(0.72 0.22 40)` - 金色/橙色，象徵「獲利與力量」
- **警示色（風險）**：`oklch(0.65 0.25 25)` - 紅色，清晰的危險信號
- **安全色（持股）**：`oklch(0.60 0.20 140)` - 綠色，象徵「生長與穩定」
- **文字主色**：`oklch(0.95 0.01 280)` - 接近白色，高對比
- **次文字**：`oklch(0.70 0.02 280)` - 淡灰，輔助資訊

### 📐 排版系統
- **Display Font**：`Poppins` 700/800 - 粗體、幾何感、現代運動風
- **Heading Font**：`Poppins` 600 - 標題層級
- **Body Font**：`Inter` 400/500 - 數據閱讀，清晰高效
- **Mono Font**：`JetBrains Mono` - 數值與程式碼，交易終端感

### 🏗️ 排版架構
- 避免居中對齊，採用**左對齊 + 卡片堆疊**
- 卡片採用**斜邊切角** (clip-path 或 border-radius 組合)
- 大量使用**垂直線條分割**，強調數據分層
- 指標卡片採用**上下分割佈局**（指標名稱 / 數值）

### ✨ 簽名元素 (Signature Elements)
1. **斜邊金色條紋** - 卡片頂部或側邊的 45° 金色斜線，象徵「上升趨勢」
2. **燈號圓形指示器** - 🟢🟡🔴 狀態燈，大尺寸（24-32px），清晰可見
3. **數據視覺化圖表** - Recharts 圖表採用金色/綠色/紅色配色，與品牌一致

### 🎬 互動哲學
- **按鈕按下**：`scale(0.96)` + 金色發光效果，160ms ease-out
- **卡片懸停**：輕微上移 + 陰影加深，傳達「可互動」
- **指標變化**：數值變化時，閃爍金色背景 (200ms)，吸引注意
- **警示彈窗**：紅色邊框 + 震動動畫 (200ms)，強制使用者注意

### 🎥 動畫指南
- **入場動畫**：從下方淡入 + 輕微上移 (300ms cubic-bezier(0.34, 1.56, 0.64, 1))
- **圖表繪製**：數據點逐一出現 (Recharts 內建 isAnimationActive)
- **數值滾動**：從舊值滾動到新值 (800ms linear)
- **警示閃爍**：背景色 0.5s 紅色 → 透明 → 紅色，重複 3 次

### 📱 排版系統
- **Display**：Poppins 700, 32-48px - 頁面標題
- **Heading 1**：Poppins 600, 24-28px - 卡片標題
- **Heading 2**：Poppins 600, 18-20px - 子標題
- **Body**：Inter 400, 14-16px - 一般文字
- **Caption**：Inter 400, 12px - 輔助資訊
- **Mono Data**：JetBrains Mono 500, 14-18px - 數值展示

### 🏢 品牌本質
**一句定位**：為積極型台股波段交易者打造的「風控決策儀表板」，將複雜的技術分析與籌碼數據轉化為清晰的交易信號，幫助交易者在高檔融資過熱時保持冷靜、在低檔支撐時果決加碼。

**三大人格特質**：
1. 🔥 **犀利** - 直言不諱，不迴避風險警示
2. 💪 **專業** - 基於科學指標，絕不感情用事
3. ⚡ **即時** - 實時數據、快速決策、果決執行

### 💬 品牌聲調 (Brand Voice)
**標題 + CTA 風格**：
- ✅ 「你的單早已經被監控了！高檔融資暴衝量大不漲，不要進去送人頭！」
- ✅ 「ATR 防守線已跌破，立即落袋為安！」
- ✅ 「現金不足 10%，嚴禁一次性歐印！」
- ❌ 避免：「歡迎來到我們的應用」、「開始您的交易之旅」（過於溫和）

### 🎯 標誌與品牌色
- **Logo**：一個粗體的「💪」肌肉圖示 + 金色邊框，象徵「力量與決斷」
- **Favicon**：同上，32x32px
- **簽名色**：`oklch(0.72 0.22 40)` - 金色，在所有重要元素上出現
