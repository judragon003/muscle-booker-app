# 🏷️ Triage Label 標籤規範

本專案使用標準 5 大 Triage 標籤，供 `/triage` 與 `/to-tickets` 技能維護任務狀態：

| 標籤名稱 (Label) | 用態與含義 |
| :--- | :--- |
| **`needs-triage`** | 待評估分流的新 Issue / 需求 |
| **`needs-info`** | 需要使用者補充資訊或澄清細節 |
| **`ready-for-agent`** | 規格明確，可直接交由 AI Agent 執行開發 |
| **`ready-for-human`** | 需要人類工程師介入審查或決策 |
| **`wontfix`** | 不修復或不採納之需求 |
